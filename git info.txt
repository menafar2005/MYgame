git:  یسری اطلاعات اولیه و کلی در رابطه با گیت 

git add filename: بجای filename اسم فایل جدید رو که میخوایم به استیج اضافه کنیم می نویسیم 

git add . : همه فایل هارو یجا به استیج اضافه میکنه 

git status: وضعیت استیج رو نشون میده و اینکه رو کدوم فایل ها تغییری اعمال کردی و هنوز توی استیج ذخیره نکردی 

git commit -m "enter some massage": کامیت کردن اونایی که توی استیج هستن 

git commit -a -m "enter some message": همزمان هم فایل هارو توی استیج ذخیره میکنه و بلافاصله کامیت میکنه
